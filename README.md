## 📋 Sobre o Projeto

O **Robô do Conselho Fiscal** é um assistente virtual especializado em fiscalização condominial, treinado com o conteúdo do livro **"Conselho Fiscal 4.0"**. Diferente de qualquer outra ferramenta no mercado editorial condominial, este chatbot permite que síndicos, conselheiros e administradores tirem dúvidas em tempo real, gerem minutas de notificações e recebam orientações técnicas baseadas na legislação vigente.

### ✨ O que torna este projeto INÉDITO?

- **Primeiro chatbot condominial brasileiro** integrado a um livro físico via QR Code
- **Geração automática de pareceres fiscais** e minutas de notificação
- **Respostas baseadas 100% no conteúdo do livro** + Lei 4.591/64
- **Simulação interativa** de casos reais (em desenvolvimento)
- **Zero custo de funcionamento** para o leitor

---

## 🚀 Demonstração Rápida

### Como funciona na prática:

1. O leitor escaneia o **QR Code** no rodapé de um capítulo
2. Abre o chatbot no celular ou computador
3. Digita uma pergunta como: *"Como analisar um balancete com recibos não identificados?"*
4. O robô responde em segundos, **citando o capítulo e a página do livro**
5. Se solicitado, **gera automaticamente uma minuta de notificação** para o síndico
